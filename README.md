# Student Portfolio and Academic Management Website

## COS 106 вЂ“ Introduction to Web Technologies

### Project Overview

This project was developed as part of the COS 106 Practical Term Project at MIVA Open University.

The website serves as a personal academic portfolio that showcases a student's educational background, technical skills, projects, and academic activities while demonstrating the use of HTML5, CSS3, and JavaScript.

The website follows responsive web design principles and includes interactive features such as an academic planner and contact form validation.

# Features

## Home Page

- Professional landing page
- Student profile photograph
- Welcome message
- Brief biography
- Skills showcase
- Academic highlights table
- Introduction video

## About Me

- Personal profile
- Educational background
- Career aspirations
- Technical skills
- Hobbies and interests
- Personal quote

## Projects

- Academic project showcase
- Project descriptions
- Technologies used
- Screenshots
- External project links

## Academic Planner

Users can:

- Add academic tasks
- Mark tasks as completed
- Delete tasks
- Manage daily study activities using JavaScript

## Contact Page

Interactive contact form with JavaScript validation.

Validation includes:

- Required fields
- Name validation
- Email validation
- Phone number validation
- Message validation

# Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
